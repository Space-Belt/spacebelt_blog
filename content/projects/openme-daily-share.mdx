## 프로젝트 개요

OPENME는 일상을 공유하는 React Native Expo 기반 개인 프로젝트입니다. 사용자가 가입하고 글을 작성하며 댓글로 소통하는 커뮤니티 앱의 기본 흐름을 직접 구현했습니다.

## 맡은 부분

Firebase Auth와 Firestore를 활용해 인증, 게시판, 댓글, 대댓글 기능을 구현했습니다. 서버를 직접 운영하지 않는 환경에서 Firebase를 이용해 앱의 핵심 사용자 흐름을 빠르게 구성했습니다.

## 구현 포인트

- Firebase Auth 기반 로그인
- Firestore 게시글 저장과 조회
- 댓글 / 대댓글 데이터 구조
- 커뮤니티 앱의 기본 사용자 흐름

## 배운 점

앱의 기본 기능은 단순해 보여도 데이터 구조를 어떻게 잡느냐에 따라 이후 기능 확장이 달라졌습니다. 댓글과 대댓글을 구현하며 화면 구조와 데이터 구조를 함께 설계해야 한다는 점을 배웠습니다.
