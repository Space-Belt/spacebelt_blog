## 프로젝트 개요

똑구는 구독 관리 서비스입니다. 앱과 웹 화면을 담당했고, OCR을 활용한 영수증 분석 기능과 행동 데이터 추적 작업을 진행했습니다.

## 맡은 부분

React Native Expo 기반 앱과 웹 화면을 담당했습니다. OCR로 영수증 정보를 분석하고, Mixpanel과 Firebase Analytics를 활용해 사용자의 주요 행동을 추적할 수 있도록 이벤트를 설계했습니다.

## 구현 포인트

- OCR 영수증 분석 흐름
- 구독 관리 화면
- Mixpanel 이벤트 설계
- Firebase Analytics 행동 추적
- 앱과 웹의 화면 흐름 정리

## 배운 점

기능을 만드는 것과 사용자가 실제로 어떻게 쓰는지 확인하는 것은 다른 일이었습니다. 데이터 추적을 붙이면서 화면 구현 이후의 판단 기준까지 생각하게 되었습니다.
