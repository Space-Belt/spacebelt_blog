## 프로젝트 개요

똑구는 사용자의 구독 정보를 관리하는 서비스입니다. 앱과 웹 프론트엔드 화면을 담당했고, 영수증을 분석해 구독 정보를 연결하는 OCR/AI 기반 기능과 사용자 행동 데이터 추적을 구현했습니다.

## 맡은 부분

React Native Expo 기반 앱과 웹 화면을 작업했습니다. OCR로 영수증 정보를 읽고 AI 분석 결과를 서비스 데이터와 연결하는 흐름을 구성했으며, Mixpanel과 Firebase Analytics로 주요 행동 이벤트를 추적할 수 있게 정리했습니다.

## 구현 포인트

- OCR/AI 기반 영수증 분석 흐름
- 구독 관리 화면
- Mixpanel 이벤트 추적
- Firebase Analytics 행동 데이터 기록
- 앱과 웹의 화면 흐름 정리

## 배운 점

기능을 구현하는 것과 사용자가 실제로 어떻게 쓰는지 확인하는 일은 달랐습니다. 분석 기능과 행동 데이터 추적을 함께 붙이면서 화면 구현 이후의 운영 판단 기준까지 고려하게 되었습니다.
