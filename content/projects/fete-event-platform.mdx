## 프로젝트 개요

FETE는 이벤트 정보를 찾고, 등록하고, 판매하고, 입장 인증까지 처리하는 이벤트 플랫폼입니다. 앱과 관리자 웹을 함께 다루며 화면과 기능을 구현했습니다.

## 맡은 부분

React Native CLI 기반 앱 구조를 다루고, New Architecture 기반 구조를 경험했습니다. 광고 식별자 수집을 위해 IDFA와 GAID를 커스텀 TurboModule로 연결하고 Codegen 세팅을 진행했습니다.

<Callout>
화면 구현뿐 아니라 네이티브 설정과 연결도 같이 확인해야 하는 프로젝트였습니다.
</Callout>

## 구현 포인트

- 이벤트 정보 탐색과 검색 흐름
- 이벤트 등록과 판매 관리 흐름
- 입장 인증과 운영 관리 화면
- New Architecture 기반 React Native 구조 정리
- IDFA / GAID 커스텀 Native Module 구성

## 배운 점

React Native 앱은 JavaScript 화면만으로 완성되지 않았습니다. 네이티브 권한, 빌드 설정, Codegen, 운영 화면까지 연결해야 실제 서비스가 되었습니다.
